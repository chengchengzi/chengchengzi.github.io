Python学习笔记

coding
  • python

说明

学习的网站是廖雪峰的Python教程，挺好的。 一直用Onenote记笔记，但是不知道怎么生成目录，还是写成博客记吧。

一、python简介

python适合开发哪些类型的应用？

- 网络应用-网站、后台服务等;
- 日常需要的小工具-脚本任务等;
- 把其他语言开发的程序再包装起来，以便使用;

python的优缺点：

  • 优点：

    • 提供了非常完善的基础代码库，包括网络、文件、GUI数据库、文本等大量内容;
    • 简单易懂；
    • 跨平台，可以预习再Windows\Mac 和各种Linux/Unix系统上。
  • 缺点：
    • 速度慢，因为python是解释型语言，会比c\c++慢一些;

二、输入输出

输出print()

  • 可以输出一个字符串或者多个字符串，用逗号“，”隔开

  • 打印整数或者计算结果

三、数据类型

  • 在Python中，能够直接处理地数据类型有：整数、浮点数、字符串、布尔值、空值，还提供了列表、字典等多种数据类型，还有自定义数据类型;
    -Python允许用’’’…’’’地格式表示多行内容

  • 空值是Python里地一个特殊地值，用None表示。None不能理解为0，None是一个特的地空值。

  • 变量名必须是大小写英文、数字和下划线 _ 的组合，且不能用数字开头
  • Python用的是动态语言
  • 在Python中，通常用全部大写的变量表示常量
  • 除法：/ 除法计算的结果是浮点数，// 地板除 除法只取结果的整数部分，%除法 是整除求余运算

字符串编码

  • Python 3的字符串使用Unicode，直接支持多种语言
  • 按照UTF-8编码读取源代码，# -- coding: utf-8 --
  • 在Python中，采用的格式化方法和c语言是一致的，用%实现

list、tuple、dict、set

  • dict和set的key必须是不可变对象。不可变对象指调用对象自身的任意方法，也不会改变对象自身的内容，反而会创建新的对象并返回。
  • dict特点：
    • ① 查找和插入的速度极快，不会随着key的增加而增加（用空间来换取时间）
    • ② 需要占用大量的内存
  • ist特点：
    • ① 查找和插入的时间随元素的增加而增加
    • ② 占用空间小，节省空间
  • set是无重复无序元素的集合
  • set和dict的唯一区别是在于有没有存储对应的value

四、函数

返回值

  • Python函数返回多个值实际上是返回一个tuple
  • 函数体内部可以用return随时返回函数结果
  • 函数执行完毕没有return语句时，自动return None

参数

  • Python参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数、关键字参数
  • 设置默认参数时，必选参数在前，默认参数在后
  • 默认参数必须指向不变对象

  • 在参数前面加一个 * 号，调用该参数时就可以传入任意个参数了。包括0个参数,可变参数在函数调用时自动组装为一个tuple

  • 关键字参数允许传入0个或任意个参数，在函数内部自动组装为一个dict

  • *args是可变参数，args接收的是一个tuple，**kw是关键字参数，kw接收的是一个dict，*args和** kw是Python的习惯写法

  • 命名关键字参数需要一个特殊分隔符 * ，*后面的参数被视为命名关键字参数，即只接受对于名字的参数，调用时必须传入参数名

五、函数的高级特性

  • list、tuple、字符串都可以进行切片操作，倒数第一个元素的索引是-1
  • 生成器generator：一边循环一边计算
  • 普通函数和generator函数区别：调用普通函数直接返回结果，调用generator实际返回一个generator对象
  • 凡是可作用于for循环的对象都是Iterble类型，凡是可作用于next（）函数的对象都是Iterator类型，其表示一个惰性计算序列。
  • 集合数据类型如list、dict、str等都是Iterable，但不是Iterator。

六、函数式编程

  • Python对函数式编程提供部分支持，由于其允许使用变量，因此并不是纯函数式编程语言；
  • 函数式编程就是一种抽象程度很高的编程范式，允许把函数本身作为参数传入另一个函数，还允许返回一个函数；
  • 高阶函数：
  • map() 接收两个参数，一个是函数，一个是Iterable，map将传入的函数依次作用到序列的每 个元素，并把结果作为新的Iterator返回；
  • Reduce()把一个函数作用在一个序列上[x1,x2,x3,…]上，这个函数必须接收两个参数，reduce 把结果继续和序列的下一个元素做累计运算；
  • Filter()接收一个函数和一个序列，filter()把传入的函数依次作用于每个元素，然后根据返回值是True还是False决定保留还是丢弃该元素
  • sorted(list，key，reverse = True)，关键字Key指向映射函数，作用于list上的每一个元素，然后sorted对Key作用后的list进行排序，reverse 为true表示反向排序。
  • 匿名函数 lambda x：x * x ，冒号前的x表示函数参数，冒号后面的表达式就是匿名函数的返回值
  • 装饰器
    • 定义：在代码运行期间动态增加功能的方式，称之为装饰器；
    • 本质上，decorator是一个返回函数的高阶函数；
  • 偏函数
    • 当函数的参数个数太多，需要简化时，使用functools.partial可以创建一个新的函数，这个新函数可以固定住原函数的部分参数，从而在调用时更简单。

七、模块

  • Python模块的标准文件模板：

  • 作用域

    • 正常的函数和变量名都是公开的，可以直接被引用，Python通过_ 前缀来实现有些函数和变量我们仅仅在模块内部使用，即表明是私有的，类似于_xxx和__xxx这样的函数或变量就是私有的，不可被直接引用；
    • 在Python中安装第三方模块，是通过包管理工具pip完成的

八、面向对象

  • 面向对象的三大特性：封装、继承、多态
  • Python的构造函数：init（self，param1，param2）其第一参数永远是self，表示创建的实例本身，有了init方法，在创建实例的时候，就必须传入与之匹配的参数，但self不需要传；
  • 私有变量，在变量前加两个下划线 “__xxxx；
  • 多态的威力：调用方只管调用，不用管细节；满足开闭原则，即对扩展开放，对修改封闭
  • 动态语言的“鸭子类型”决定不要求严格的继承体系，一个对象只要“看起来像鸭子，走路像鸭子”，那它就可以被看做是鸭子。Pyhton的“file-like-object”就是一种鸭子类型；
  • 获取对象的信息
  • 总是优先使用isinstance()判断类型
  • dir()函数可以获得一个对象的所有属性和方法，它返回一个包含字符串的list;
  • getattr()、setattr()、hasattr()可以直接操作一个对象的状态；
  • 写程序时。不要对实例属性和类属性使用相同的名字，因为相同名称的实例属性将屏蔽掉类属性，但是删除实例属性后，再使用相同的名称，访问到将是类属性；

九、面向对象高级编程

  • 动态语言的灵活性:创建了一个class的实例后，我们可以给该实例绑定任何属性和方;
  • 特殊的slots变量，来限制该class实例能添加的属性, but定义的属性仅对当前类实例起作用，对继承的子类是不起作用的;
  • python内置的 @property装饰器负责把一个方法变成属性调用
  • 不定义setter方法就是一个只读属性
  • callable()函数，我们就可以判断一个对象是否是“可调用”对象
  • @unique装饰器可以帮助我们检查保证没有重复值

十、错误，编译和调试

  • 只有在必要的时候才定义我们自己的错误类型。如果可以选择Python已有的内置的错误类型（比如ValueError，TypeError），尽量使用Python内置的错误类型。
  • 调试的方式
    • 打印，print（）；
    • 断言，assert xxx,yyy，如果xxx为假，则输出yyy；
    • 日志，logging，信息的级别有：debug，info，warning,error；
    • 调试器pdb，pdb.set_trace() 设置一个断点；
    • 支持调试功能的IDE设置断点，调试

十一、IO编程

  • 输入输出流，Input Stream就是数据从外面（磁盘、网络）流进内存，Output Stream就是数据从内存流到外面；
  • 解决CPU、内存的速度和外设的速度不匹配的方法：同步IO、异步IO，二者的区别在于CPU是否等待IO的指向结果；
  • 在Python中，文件读写是通过open()函数打开的文件对象完成的；
  • with 可以自动调用close()，使用with语句操作文件IO是个好习惯；
  • StringIO和BytesIO是在内存中操作str和bytes的方法，使得和读写文件具有一致的接口；
  • 序列化：将变量从内存中变成可存储或传输的过程称之为序列化，使用json模块可以使序列化更通用、更符合Web标准；
  • 为什么要序列化：因为要在不同的编程语言之间传递对象，这就就必须把对象序列化为标准格式。

十二、进程和线程

  • 多任务的实现有3种方式：多进程模式； 多线程模式； 多进程+多线程模式。；
  • 在Unix/Linux下，可以使用fork()调用实现多进程；
  • 要实现跨平台的多进程，可以使用multiprocessing模块；
  • 进程间通信是通过Queue、Pipes等实现的；
  • ThreadLocal解决了参数在一个线程中各个函数之间互相传递的问题；
  • 分布式进程： Python的multiprocessing模块不但支持多进程，其中managers子模块还支持把多进程分布到多台机器上；

十三、正则表达式

  • 正则表达式的作用是匹配字符串；
  • 正则表达式中如何用字符来描述字符，可以在python的官网查询到；
    https://docs.python.org/3/library/re.html?highlight=re#
  • Python提供re模块，包含所有正则表达式的功能。

十四、Python常用的内置模块：

  • datetime、collections、based64、struct、hashlib、hmac、itertools、contextlib、
    urllib、XML、HTMLParser；
  • time 如果要存储datetime，最佳方法是将其转换为timestamp再存储，因为timestamp的值与时区完全无关；
  • collections模块提供了一些有用的集合类，可以根据需要选用；
  • Base64是一种任意二进制到文本字符串的编码方法，常用于在URL、Cookie、网页中传输少量二进制数据；
  • Python提供了一个struct模块来解决bytes和其他二进制数据类型的转换；
  • hashlib提供了常见的摘要算法，如MD5，SHA1等等,摘要算法不是加密算法，不能用于加密（因为无法通过摘要反推明文），只能用于防篡改；
  • Python内置的hmac模块实现了标准的Hmac算法，它利用一个key对message计算“杂凑”后的hash，使用hmac算法比标准hash算法更安全，因为针对相同的message，不同的key会产生不同的hash；
  • 内建模块itertools提供了非常有用的用于操作迭代对象的函数；
  • urllib提供了一系列用于操作URL的功能，操作XML有两种方法：DOM和SAX。DOM会把整个XML读入内存，解析为树，因此占用内存大，解析慢，优点是可以任意遍历树的节点。SAX是流模式，边读边解析，占用内存小，解析快，缺点是我们需要自己处理事件。正常情况下，优先考虑SAX，因为DOM实在太占内存；
  • 利用HTMLParser，可以把网页中的文本、图像等解析出来；

十五、常用第三方模块

  • pillow、requests、chardet、psutil
  • PIL pillow 提供了操作图像的强大功能，可以通过简单的代码完成复杂的图像处理。要详细了解PIL的强大功能，请参考Pillow官方文档：https://pillow.readthedocs.org/ ；
  • requests是一个Python第三方库，处理URL资源特别方便；
  • chardet用于检测编码，获取到编码后，再转换为str，就可以方便后续处理
  • Python程序利用psutil使得获取系统信息。psutil还可以获取用户信息、Windows服务等很多有用的系统信息，具体请参考psutil的官网：https://github.com/giampaolo/psutil

十六、GUI和网络编程

  • Python内置的Tkinter可以满足基本的GUI程序的要求，如果是非常复杂的GUI程序，建议用操作系统原生支持的语言和库来编写;
  • 网络编程就是如何在程序中实现两台计算机的通信；
  • TCP用TCP协议进行Socket编程在Python中十分简单，对于客户端，要主动连接服务器的IP和指定端口，对于服务器，要首先监听指定端口，然后，对每一个新的连接，创建一个线程或进程来处理。通常，服务器程序会无限运行下去。同一个端口，被一个Socket绑定了以后，就不能被别的Socket绑定了；
  • UDP的使用与TCP类似，但是不需要建立连接。此外，服务器绑定UDP端口和TCP端口互不冲突，也就是说，UDP的9999端口与TCP的9999端口可以各自绑定。

十七、Web编程

  • ORM技术：Object-Relational Mapping，把关系数据库的表结构映射到对象上；
  • 方法：GET还是POST的区别，GET仅请求资源，POST会附带用户数据；
  • HTML定义了页面的内容，CSS来控制页面元素的样式，而JavaScript负责页面的交互逻辑。
