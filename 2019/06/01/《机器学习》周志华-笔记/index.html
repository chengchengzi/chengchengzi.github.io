第二章 模型评估与选择

误差

  • 训练误差：训练集上产生的误差
  • 测试误差：测试集上产生的误差
  • 泛化误差：对新样本的误差
  • 一般用测试误差来估计泛化误差

如何从数据集D中产生训练集S和测试集T？

  • 1.留出法：将D划分为两个互斥的集合，即$D = S \cup T, S \cap T = \emptyset$。
    • 留出法缺点：S和T的大小会对评估结果有影响，若$S > T$，模型不够稳定，方差较大，容易过拟合；若$S < T$，降低评估结果的保真性，偏差较大，容易欠拟合。一般取2/3 ~ 4/5 的样本用于训练，剩余的样本用于测试。
  • 2.交叉验证法：将D划分为k个大小相似的互斥子集，即$D=D_1 \cup D_2 \cup D_3 … \cup D_k, D_i \cap D_j = \emptyset, i \neq j$。每次用k-1子集的并集作为训练集，剩余的一个子集作为测试集。进行k次训练和测试，取k次测试的均值作为测试结果。
    • 交叉验证法缺点：评估结果的稳定性和保真性受控于k的取值，k常取10。
  • 3.留一法：交叉验证法的特例，D中包含m个样本，取k=m。
    • 留一法优点：评估结果较准确
    • 留一法缺点：D较大时，计算开销比较大。
  • 4.自助法：有放回采样，随机从D中抽取一个样本，放入训练集S中，然后放回D中，重复执行m次，测试集T = D / S。
    • 自助法优点：在D较小时，难以有效划分S/T时很有用
    • 自助法缺点：引入了估计偏差，有放回采样产生的S，改变了初始数据集的分布。
  • 5.当D足够时，留出法和交叉验证法更常用。
  • 6.在训练集训练模型，在验证集进行模型选择和调参，测试集测试泛化性能。

性能度量

  • 错误率：$E = a \div m$，m个样本有a个样本分类错误；
  • 精度 = 1 - 错误率 = $1 - E = (1-a \div m) * 100%$
  • 查准率P 和 查全率R
    $$P = TP \ div TP + FP$$

第四章 决策树

思想

决策树是基于树结构来进行决策的，学习的目的是产生一颗泛化能力强的决策树。一般的，一颗决策树包含一个根结点、若干个内部结点和若干个叶结点；叶结点对应于决策边界，其他每个结点则对应于一个属性测试。

选择最优划分属性的方式

  • 信息增益：选择信息增益最大为划分属性；
  • 增益率：选择增益率最大的作为划分属性；
  • 基尼指数：选择基尼指数最小的属性作为划分属性。
